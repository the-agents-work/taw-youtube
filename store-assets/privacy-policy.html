TAW YouTube — Privacy Policy

Last updated: 2026-05-23

Summary. TAW YouTube does not collect, store, or sell personal data. Your OpenAI API key and preferences are stored locally in Chrome extension storage. Captions or audio are sent to OpenAI only when you start a translation session.

Data processed

  • OpenAI API key, stored locally and sent only to api.openai.com.
  • YouTube captions in Smart mode, or audio from the active video in Realtime and Standard modes, sent to OpenAI for translation and speech generation.
  • Local settings such as language, voice, tier, volume, and overlay layout.

Data not collected

  • No telemetry, analytics, crash reports, or usage statistics.
  • No browsing history, video titles, YouTube account data, cookies, or tracking IDs.

Permissions

  • activeTab and scripting let the extension run on the YouTube tab you invoke.
  • storage remembers your API key and preferences locally.
  • webRequest observes YouTube caption requests for subtitle-first translation.
  • Host permissions are limited to YouTube and OpenAI API domains.
