TAW YouTube — Privacy Policy

Last updated: 2026-05-23

TL;DR. TAW YouTube does not collect, store, or sell personal data. Your OpenAI API key stays on your device. Audio you choose to translate is sent to OpenAI only to produce transcription, translation, and speech.

What TAW YouTube is

TAW YouTube is a Chrome extension that translates the YouTube video you are watching. Smart captions mode translates existing YouTube captions; Realtime and Standard modes can process audio for voice-over. It runs in your browser and has no app server.

What data is processed

  • Your OpenAI API key. Stored locally in chrome.storage.local with TRUSTED_CONTEXTS access level. The key is sent only to api.openai.com for requests you start.
  • YouTube captions and audio. Smart captions mode fetches caption text and timestamps. Realtime and Standard modes capture audio from the active video using HTMLMediaElement.captureStream(). Captions and audio are sent to OpenAI only for translation and speech generation.
  • Overlay layout. Panel position and size are stored in YouTube-origin localStorage. This is cosmetic only.
  • Settings. Tier, language, voice, and volume preferences are stored locally in Chrome extension storage.

What is not collected

  • No telemetry, analytics, crash reports, or usage statistics.
  • No browsing history, video titles, URLs, or YouTube account data.
  • No cookies, advertising IDs, fingerprints, or tracking pixels.

Third-party processor

OpenAI processes the audio, transcript text, translation text, and generated speech needed for the feature. Provider-side handling follows OpenAI's policies and your OpenAI account settings.

Permissions

  • activeTab, scripting — inject the content script and overlay when you press Start.
  • storage — remember your API key and preferences.
  • webRequest — observe YouTube caption requests so subtitle-first mode can reuse signed caption URLs.
  • Host permissions for youtube.com and api.openai.com.

Your choices

  • Stop a session at any time with the Stop button or by closing the tab.
  • Remove your API key from the popup or uninstall the extension to wipe locally stored data.
